### General principles
The chat widget sends and receives the messages of the types listed below. It also provides the user interface to send and receive messages.  

### Message types:  

* text;
* attached image;
* attached file;
    * text with a quote, where the quote can be as follows:
    * text;
    * attached image;
    * attached file.
